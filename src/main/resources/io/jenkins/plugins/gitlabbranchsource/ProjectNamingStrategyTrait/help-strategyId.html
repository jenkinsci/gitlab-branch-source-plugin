Determines the naming convention of projects
Full Project Path
This option will use the full project path for the group and project. Example: my-group/my-project my-group/other-project my-group/my-subgroup/third-project
Project Name
This option will use project names from GitLab project name. Example: My Project Other project My Subgroup / Third project
Contextual Project Path
This option will use the project path with nested groups and project, but not the owner segment. Example: my-project other-project my-subgroup/third-project
Simple Project Path
This option will use the project path with no groups. Example: my-project other-project third-project
